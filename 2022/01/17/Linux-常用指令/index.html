Linux 常用指令

  • Tab : 补全

  • cd : 改变目录

  • cd . . 回退到上一个目录，直接cd进入默认目录

  • pwd : 显示当前所在的目录路径。

  • ls: 列出当前目录中的所有文件。

    • ls -l 可以查看文件的详细属性
    • Windows可使用 dir 指令

  • touch : 新建一个文件 如 touch index.js 就会在当前目录下新建一个index.js文件。

  • rm: 删除一个文件, rm index.js 就会把index.js文件删除。

    • rm -r : 删除一个文件夹, rm -r src 删除src目录

  • mkdir: 新建一个目录,就是新建一个文件夹。

    • mkdir wangkr —— 新建一个wangkr文件夹

  • mv 移动文件, mv index.html src

  • reset 重新初始化终端/清屏。

  • clear 清屏。

  • history 查看命令历史。

  • help 帮助。

  • exit 退出。

  • #表示注释

  • wget：下载文件的工具

  • Vim 打开文件：vim /test/vi.txt

  • 执行shell脚本


Anaconda的使用

  • 创建环境

    conda create -n envname python=3.7

  • 激活虚拟环境

    • linux： source activate envname
    • windows：activate envname

  • 查看已有的虚拟环境

    conda env list

  • 安装第三方库

    • pip install XXX
    • pip install XXX.whl（官网下载whl文件）

  • 退出虚拟环境

    • Linux：source deactivate envname
    • Windows：deactivate envname，也可以使用 activate root 切回 root 环境

  • 删除虚拟环境

    conda remove -n envname –all
