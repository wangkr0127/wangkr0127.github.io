人工神经网络 ANN

人工神经网络 ANN

联结主义学派：又称仿生学派，认为人的思维基元是神经元，而不是符号处理过程

核心：智能的本质是联接机制

原理：神经网络及神经网络间的连接机制和学习算法

人工神经网络：基于模仿生物大脑的结构和功能而构成的一种信息处理系统

自底向上，起源于生物神经系统 ，从 结构模拟 到 功能模拟

生物里的神经元：

  • 兴奋状态：对输入信息整合后使细胞膜电位升高，当高于动作电位的阈值时，产生神经冲动，并由轴突输出。
  • 抑制状态：对输入信息整合后使细胞膜电位降低，当低于动作电位的阈值时，无神经冲动产生。

神经元之间的柔性连接：突触的信息传递特性是可变的——学习记忆的基础


神经元模型

  • 二值神经元模型——MP模型（组合函数：加权和函数；激活函数：阈值函数）
  • 模拟神经元模型（组合函数：加权和函数；激活函数：Sigmod函数等）

M-P模型

神经网络学的是 连接权 和 阈值

多个人工神经元按照特定的网络结构联接在一起，就构成了一个人工神经网络。

神经网络的目标就是将输入转换成有意义的输出。


ANN的学习规则

基于能量最小原则：保持突触强度需要能量，所以在需要的地方保持，在不需要的地方不保持。

ENERGY = measure of task performance error （可以用性能评价作能量的函数定义）


ANN的2种结构

  • 前馈结构(Feedforward Architecture) ：无环、静态

​ （前馈无需考虑时间因素）

  • 反馈/循环结构(Feedback/Recurrent Architecture) ：有环、动态

​ （反馈输出会变成输入）


ANN的学习方法

通过神经网络所在环境的模拟过程，调整网络中的自由参数（通过数据集的数据进行学习）

学习策略: Error Correction（错误纠正）

最小化实际输出与期望输出之间的误差(Supervised)

常用的错误纠正策略：Delta Rule 、BP学习算法


感知机

感知机由两层神经元构成：输入层 + 输出层，输出层是M-P神经元，只有输出层进行激活函数处理

主要针对监督学习，感知机是最简单的前馈网络

Loss function 损失函数：预测值和实际值之间的差距度量（一般学习目标是最小化损失函数）

感知机实质上是一种神经元模型，与M-P模型的区别：感知机组合函数里的权值w是可变的

感知机特性：

通过感知机可以实现与、或、非，但是不能实现异或

感知机收敛性：

感知机收敛定理 ：若训练数据集是线性可分的，则感知机模型收敛。

感知机存在的问题
  • 噪声（不可分情况）:数据集不可分，感知机无法收敛（无法求得最终解）
  • 泛化性：无法找到性能最好的超平面

改进感知机：加入隐层可实现异或操作

多层感知机 (MLP)

由 输入层、隐层、输出层 构成

层间神经元全连接，采用模拟神经元模型

多层感知机就是一个多个隐层的全连接神经网络

MLP的表达能力
  • 3 layers: 可以模拟所有的连续函数
  • 4 layers: all functions
BP网络：BP算法训练的多层前馈神经网络（多层感知机），采用 Sigmoid 激活函数

多层感知机的学习算法：BP学习算法

BP算法步骤：

Step1. Select a pattern from the training set and present it to the network.

Step2. Compute activation of input, hidden and output neurons in that sequence.

Step3. Compute the error over the output neurons by comparing the generated outputs with the desired outputs.

Step4. Use the calculated error to update all weights in the network, such that a global error measure gets reduced.

Step5. Repeat Step1 through Step4 until the global error falls below a predefined threshold.

调整权重的过程采用：梯度下降法

梯度下降法——寻找最小化均方误差e(w)的权重w (优化方法)

多层感知机的优缺点

  • 优点：很强的表达能力；容易执行

  • 缺点：

    • 收敛速度慢——采用Newton法
    • 过拟合（Over-fitting）—— 加正则化项，约束权值的平滑性，采用更少（但足够数量）的隐层神经元
    • 局部极小——尝试不同的初始化增加扰动

深度神经网络训练

  • 自下向上的非监督学习（贪婪预训练获得网络良好的初始化 ）
  • 自顶向下的监督学习
    • 通过带标签的数据去训练，误差自顶向下传输，对网络进行微调
    • 微调特征（中间层），使得与问题更相关

Deep Learning 的常用模型

AutoEncoder自动编码器

学习过程：

  • 无标签数据，用非监督学习学习特征
  • 监督学习：Deep结构，有监督微调

三层结构：输入层，隐藏层，输出层

  • 输入层神经元数=输出层神经元数

  • 隐层神经元数量<输入层神经元数量 意义：迫使隐藏层节点学习得到输入数据的压缩表示方法

两隐层自编码网络MNIST手写数字识别

  1. 用原始输入x(k)训练第一个自编码器，学习得到原始输入的一阶特征表示h(1)(k)
  2. 把上一层的一阶特征作为另一个稀疏自编码器的输入，使用它们来学习二阶特征h(2)(k)
  3. 将二阶特征作为softmax分类器的输入，训练得到一个能将二阶特征映射到数字标签的模型
  4. 将这三层结合起来构成一个栈式自编码网络，通过反向传播算法(BP)同时调整所有层的参数以改善学习结果(称为整体细调fine- tuning)

栈式自编码器神经网络

学习方式：前层的输出作为下一层输入的方式依次训练

如果网络的输入数据是图像，网络的第一层会学习如何去识别边，第二层一般会学习如何去组合边，从而构成轮廓、角等。更高层会学习如何去组合更形象且有意义的特征。

如果输入数据集包含人脸图像，更高层会学习如何识别或组合眼睛、鼻子、嘴等人脸器官。


Hopfield Network

单层全互连、对称权值的反馈网络

网络演化：从初始状态按能量减小的方向进行演化，直到达到稳定状态。稳定状态即为网络的输出


波尔兹曼机 Boltzmann Machine

属于反馈神经网络
BM基本原理：
  • BM中一部分神经元与外部相连，可以起到网络的输入、输出功能，或者严格地说可以受到外部条件的约束。另一部分神经元则不与外部相连，因而属于隐单元
  • 每个神经元只取1或0这两种状态：状态1代表该神经元处于接通状态，状态0代表该神经元处于断开状态

受限波尔兹曼机 Restricted Boltzmann Machines（RBM）

属于反馈神经网络
  • 一个可见层
  • 一个隐层
  • 层内无连接
  • 学习目标：极大似然

Deep Belief Networks(DBN) 深度置信网络

概率生成模型，由多个RBM堆积而成（栈式叠加）

深层结构——多层

非监督的预学习提供了网络好的初始化

监督微调(fine-tuning)

DBN 网络结构：

DBN 学习算法：

  • 非监督的预学习+监督微调
  • 训练的时候采用逐层贪婪训练使得网络层数堆积

Deep Boltzmann Machines 深度玻尔兹曼机

所有层间无向连接 （同层神经元间无连接）

高层表示由无标注数据建立

标注数据仅用来微调网络

DBM 学习算法：

  • 非监督的预学习+监督微调
