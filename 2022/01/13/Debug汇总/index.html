1、通过官网或者 Github 链接直接下载 tar 或者 whl 文件

2、pip install XXX.whl / XXX.tar.gz 直接安装（同一路径下）



python安装第三方包的安装路径

  • dist-packages：系统自带的python
  • site-packages：自己安装的python


python使用os库的注意事项

os.mkdir 用来创建缺失的目录（只能创建最后一级）：

如果目录有多级，则创建最后一级，如果最后一级目录的上级目录有不存在的，则会抛出一个 Error。
