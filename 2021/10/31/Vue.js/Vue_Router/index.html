动态路由匹配：

我们经常需要把某种模式匹配到的所有路由，全都映射到同一个组件。例如，我们有一个 User 组件，对于所有 ID 各不相同的用户，都要使用这个组件来渲染。那么，我们可以在 vue-router 的路由路径中使用 动态路径参数 来达到这个效果：

此时，形如 /user/foo 和 /user/bar 都将映射到相同的路由，都使用 User 组件进行渲染。

  • 通过 $route.params.id 能获取 path 参数
  • 最理想的路由监听方法：watch —— 可对路由变化作出响应

详情见：https://router.vuejs.org/zh/guide/essentials/dynamic-matching.html
