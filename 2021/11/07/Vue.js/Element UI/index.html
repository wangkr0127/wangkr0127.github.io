Element UI

Element UI 是一套采用 Vue 2.0 作为基础框架实现的组件库

官方文档：https://element.eleme.cn/#/zh-CN/component/installation

项目案例：https://github.com/oinsd/Vue.js-Learning-Example

​ （NO.15_简介_简单组合效果_布局与容器）

（样例的项目不包含 node_modules，需要 npm install 进行导包）


样例项目中几个单文件组件的介绍：

  • App_Layout_Container.vue：一共24列，偏移4列开始，使用16列
  • App_form_2.vue：一共24列，偏移8列开始，使用8列

注：想要在某个方法里通过 this 寻找某个 data 属性值，可以先用 console.log(this) 在控制台输出组件对象信息，再寻找该 data 属性，进而完成 this 的访问
