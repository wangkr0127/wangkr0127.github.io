Vue简单本地应用--记事本

可以根据网页模板 + F12 查看 Elements 中的 HTML 源码分析模板中各个组件的布局情况，从而更好的进行前端页面开发。

Summarize

  • 列表结构可以通过 v-for 指令结合数据生成

  • v-on 结合事件修饰符可以对事件进行限制,比如 .enter

  • v-on 在绑定事件时可以传递自定义参数

  • 通过 v-model 可以快速的设置和获取表单元素的值

  • 基于数据的开发方式
