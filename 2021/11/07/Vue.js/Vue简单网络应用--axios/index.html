Vue简单网络应用--axios

axios基本使用

  • axios：功能强大的网络请求库（AJAX）
  • axios 必须先导入才可以使用
  • 使用 get 或 post 方法即可发送对应的请求
  • then 方法中的回调函数会在请求成功或失败时触发
  • 通过回调函数的形参可以获取响应内容或错误信息
