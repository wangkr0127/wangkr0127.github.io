Vue-CLI

Vue CLI 是一个基于 Vue.js 进行快速开发的完整系统，简单来说就是 Vue 搭建的架子。

教程：https://cli.vuejs.org/zh/guide/

通过 vue create 创建项目后不要修改里面文件的名称，以免之后的自动导包出错。

  • 本地运行使用 npm run serve （见README）

  • 安装依赖包使用 npm install

项目文件介绍

  • main.js 是 Vue 项目的入口文件

  • App.vue 是 root 根组件

  • components 里的 vue 是单文件组件

  • 单文件组建中用 export default 暴露该组件的 name，在另一个组件文件中用 import 进行引用

    ( 注：如果只有一个export default，将默认被认为一个名字，在main.js里面导入无论什么名字都行）

main.js:

表示 root 根组件 App 挂载到 app 上


使用 vue-ui 创建项目

创建项目的方式：

1、vue create hello-world

2、vue ui 进行网页在线项目创建 （以管理员身份打开cmd ）

创建具体步骤见：https://www.bilibili.com/video/BV1dk4y167bS?p=8


项目打包 & Nginx 发布

在 VSCode 终端键入 npm run build 进行项目的打包

配置 nginx.conf 文件进行 Nginx 发布

具体操作见：https://www.bilibili.com/video/BV1dk4y167bS?p=10

项目代码见：https://github.com/oinsd/Vue.js-Learning-Example
