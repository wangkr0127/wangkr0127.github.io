Vue组件

组件基础

组件是可复用的 Vue 实例，且带有一个名字。在这个例子中定义了一个组件 <button-counter>。我们可以在一个通过 new Vue 创建的 **Vue 根实例(根组件)**中，把这个组件作为自定义元素来使用。

组件可以理解为一个新的 元素 / 节点——提高代码复用率。

利用 Vue.component 方法创建组件。

详情见：https://cn.vuejs.org/v2/guide/components.html

组件注意事项：

  • 必须有一个根节点（new Vue），只要与内部标签不冲突。

  • 组件间可通过 props 进行通讯

  • 组件可以有多个属性，但是 data 属性的写法必须是方法+返回值


子组件和父组件实例

  • new Vue 是 root 根组件，是 abar、bbar 的父组件

  • 非单一文件组件的 CSS 样式只能放在标签属性里
  • child 为公共子组件，与 abar、bbar 同级
  • abar 里的 abar_child 是 abar 私有的子组件，只能被 abar 使用

父传子

  • 通过 props 进行通讯

  • props：接受父组件传进来的属性

  • 父组件传参给子组件使用bind绑定


子传父

使用 this.$emit 进行子传父

注意几个传递接口的 $

实例：bbar 组件传给 root 组件


ref 注册引用信息

  • ref 放标签上，拿到原生节点（元素）

  • ref 放组件上，拿到组件对象

  • 通过 $refs 进行访问


事件总线

使用 $emit 进行发送，**$on** 进行监听

空 Vue 实例，默认为中央事件总线


动态组件

动态组件就是几个组件放在一个挂载点下，然后根据父组件的某个变量来实现组件的动态切换。

固定用法：<component :is="xxx"></component> ( common 和 is 都是官方定义 )

使用 keep-alive 包裹可以保持切换之前的各种状态（切换组件是直接删掉和创建）

样例中，由于 components 的父组件是 root，所以绑定的 who 就是 root 的 data 里的 who
